PDB Point Mutation Tool A tool to perform point mutations on PDB files and
observe the change in resulting interactions

User Manual

1) Enter the four-character PDB ID in the first field.
2) Enter the Chain ID in Uppercase in the second field.
3) Enter the Residue Number in the third field.
4) Enter the Amino Acid mutation in Uppercase in the fourth field.
5) Click on Submit.

This tool has been developed using PyMol and LigPlot+ Software.
The platform allows the user to  perform point mutations on PDB files by specifying the
PDB ID, Chain ID, Residue Number, and the mutation required. 
The result page displays an image of the imported PDB and residue before and after mutation.
It also summarizes the changes in interactions produced between the mutated residue.
